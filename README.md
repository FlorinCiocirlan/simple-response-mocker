# simple-response-mocker — landing page

Landing page for **Simple Response Mocker**, a Chrome extension that intercepts `fetch` and `XMLHttpRequest` calls and returns mock responses you define in the popup.

Static site. One `index.html`. No build step.

Live: <https://mocker.ciocirlan.dev>

## Local

Open `index.html` in a browser. That's it.

## Deploy

Connected to Vercel. Pushes to `main` deploy automatically.
